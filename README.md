## üõ†Ô∏è Setup Instructions

### 1. Replace Placeholders
Update all placeholder content marked with `[brackets]`:

**Personal Information:**
- `[Your Full Name]` - Your actual name
- `[your.email@example.com]` - Your email address
- `[+1 (555) 123-4567]` - Your phone number
- `[City, State/Country]` - Your location
- `[LinkedIn URL]`, `[GitHub URL]`, `[Portfolio URL]` - Your social media links

**Content Sections:**
- Update the About/Summary section with your personal description
- Replace skill names and proficiency levels
- Add your actual education details, CGPA, and courses
- Replace project placeholders with your actual projects
- Update experience section with your work history
- Add your certifications and achievements

### 2. Add Images
Replace placeholder images:
- `profile-placeholder.jpg` - Your professional headshot (200x200px recommended)
- `project1-placeholder.jpg`, `project2-placeholder.jpg`, `project3-placeholder.jpg` - Project screenshots

### 3. Customize Colors (Optional)
The default color scheme uses a blue gradient. To change colors, update these CSS variables in `styles.css`:
- Primary gradient: `#667eea` to `#764ba2`
- Text colors: `#2c3e50` (dark), `#555` (medium)
- Background: `#f8f9fa` (light gray)

### 4. Add Your Resume PDF
- Create a PDF version of your resume
- Save it as `resume.pdf` in the same directory
- The "Download Resume" button will automatically link to it
